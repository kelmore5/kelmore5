About Me

Hey, I'm Kyle, and I'm an aspiring freelance mobile developer.

I started programming in high school and quickly grew to love everything about it. Over the past few years, I've learned at least the basics (and usually more) in over 40 different programming languages and worked with a variety of concepts, including scientific research, mobile development, and enterprise applications. I've always wanted to specialize in mobile development, and this site signifies my start to mobile exclusivity. As a freelance developer, I want to build fast, clean, and reusable applications. In other words, I uphold any app I write to be up to date with the latest standards, optimized for performance (ie I write test code for everything I write), and commented so that in the future if you decide to (unfortunately) go with someone else, he'll be able to pick up the project, no problem.
